Abstract
This project describes how to recognize certain types of human physical activities such as walking, stairs ascending/descending using 3 axis acceleration data generated by a 3 axis accelerometer being put at waist. The system was trained and tested in an experiment with multiple human subjects in real-world conditions. Several classifiers were tested using various statistical features. High-frequency and low-frequency components of the data were taken into account. We selected classifiers each offering good performance for recognizing our set of activities and investigated how to combine them into an optimal set of classifiers.
1. Introduction
Human activity recognition is an important with many application. In this project We aim to develop a model that is capable of recognizing multiple sets of daily activities like sitting,walking, climbing stairs under real-world conditions, using data collection by a single triaxial accelerometer.

1.4 Report Organization
Features were extracted from the raw accelerometer data using window size of 10 with zero overlapping between consecutive windows. window of almost five seconds having half second difference in each element can sufficiently capture cycles in activities such as walking, sitting, climbing stairs up/downstairs etc. we extracted standard features which are mean, standard deviation, autocorrelation, zero crossing etc. These feature are extracted by using Matlab only.

3. Experiments & Results
3.1 Dataset Description
The experiments have been carried out with a group of 4 volunteers within an age bracket of 19-21years. Each person performed 3 activities (SITTING, WALKING, WALKING UPSTAIRS/DOWNSTAIRS) wearing a 3 axial accelerometer on the waist that captured 3-axial linear acceleration. Notation of axis: X axis -> front / backward motion Y axis -> upward and downward motion Z axis -> left / right motion No. of dataset collected 9 Each data set includes listed 3 activities having time interval of 150 seconds for every activity. github link containing data file, feature extraction code, classifier code.
